# task-stream-records
little project: CRUD in memory com java 21, gradle, Streams, Records.


1 - This project is programmed in Java 21 and uses Records and Streams to perform CRUD operations in memory.

2 - To run the project, make sure you have Java 21 installed on your machine and gradle.

3 - import the project into eclipse kind as grdle project.

4 - config in the run configuration a new task gradle, select your workspace with project of the type gradle and put the command in first page, click in add and write then "clean", add others like a "run" or add "test".

5 - look in to console, if your code print your msg write in your code.
